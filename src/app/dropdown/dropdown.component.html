Chroma

An interactive palette tool for font and color pairings.

A lot of my favorite places to go for palettes did not have a quick way to implement in web projects.


I like to see how colors will look with font, so I think that should be a part of the selection process.

Technology

Angular2, TypeScript, WebPack.

Features in progress

SCSS/CSS Generation

Photo upload to palette

User palette creation

Full font style options

Add cards to user stack

Shareable cards

Expandable card for large typography sampling

Click to edit typography elements
